# Amazon Data Cleaning & Visualization

## Overview
This project cleans raw Amazon product data by fixing corrupted pricing, recalculating discounts, and exporting a reliable dataset. It also includes exploratory visualizations to analyze product price vs customer ratings.

## Features
- Detects and corrects pricing anomalies.
- Saves cleaned dataset as `AmazonData_Cleaned.csv`.
- Scatter plots: Price vs Rating.
- Bar charts: Average Rating by Category.

## Tech Stack
- Python (Pandas, NumPy, Matplotlib)
- Visual Studio Code
